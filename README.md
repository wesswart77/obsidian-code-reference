# Code Reference Library

A personal coding reference library for Obsidian: patterns, snippets, links, and notes tagged by language and framework.

## Features

- **New Snippet** – capture code snippets with language, framework, tags, description, and a code block
- **New Pattern** – document architecture/design patterns with categories and examples
- **Search by Tag** – find snippets and patterns by tag
- **Reference Sidebar** – browse Snippets and Patterns tabs, filterable by language
- **Settings** – configure snippets and patterns folders

## Usage

Open the command palette and search for "Code Reference" to access all commands.
